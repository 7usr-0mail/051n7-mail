## Reporting a Vulnerability

If you discover a potential security flaw or cryptographic issue within 051N7-MAIL, please do not report it publicly via GitHub Issues.

Instead, report it via encrypted GPG mail directly over Yggdrasil to:
- **GPG Identity:** `ghost@051n7.usr`
- **Target Handle:** `ghost@051n7.usr`

We acknowledge all reports within 48 hours.
